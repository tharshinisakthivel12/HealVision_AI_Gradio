## AI-Powered Diabetic Foot Ulcer Detection System

HealVision AI is an AI-powered healthcare web application that assists in the early detection of **Diabetic Foot Ulcers (DFUs)** using Deep Learning. The application analyzes uploaded wound images and provides an AI-generated diagnosis report with confidence score, risk level, and basic healthcare recommendations.

## рџ“Њ Problem Statement

Diabetic Foot Ulcers are one of the most severe complications of diabetes. Delayed diagnosis can lead to infections, hospitalization, and even limb amputation. In rural and underserved areas, immediate access to specialists is often limited.

HealVision AI provides an AI-assisted preliminary screening solution that enables users to upload a wound image and receive an instant AI-based assessment, encouraging timely medical consultation.
